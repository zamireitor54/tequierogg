Te Quiero Ogg
=================

Calendario y tarjeta personalizable para mi niña — mensajes, galería e imagen imprimible.

Este repositorio contiene una web estática (HTML/CSS/JS) con:

- Un calendario con 365 mensajes personalizados.
- Una galería de fotos que se puede ampliar y navegar.
- Una sección para añadir una imagen de certificado full-bleed.

Archivos importantes:

- `index.html` — la página principal.
- `styles.css` — estilos.
- `app.js` — lógica de la app (mensajes, calendario, galería).
- `img/` — carpeta donde colocar tus imágenes (ej. `miniña.jpg`, `img 2.jpg`, `certificate.jpg`).

Cómo usar

1. Coloca tus fotos dentro de la carpeta `img/`.
2. Abre `index.html` localmente en el navegador para probar.
3. Para publicar con GitHub Pages: ve a `Settings` → `Pages` del repo en GitHub y selecciona la rama `main` y la carpeta `/ (root)` como origen. La URL será aproximadamente:

    https://zamireitor54.github.io/tequierogg/

Consejos

- Si tus imágenes contienen espacios o caracteres no ASCII (por ejemplo `ñ`), funciona mejor renombrarlas a ASCII (`img_2.jpg`, `mini_nina.jpg`) para evitar problemas en algunos servidores.
- Si querés que renombre automáticamente o que agregue botones de descarga/imprimir para el certificado, dime y lo agrego.
